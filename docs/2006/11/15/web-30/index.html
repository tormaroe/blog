Web 3.0

onsdag 15. november 2006 Webutvikling

Er litt usikker på om denne artikkelen ikke egentlig hører hjemme i science fiction bloggen. Vi sliter fortsatt med å bli enige om hva Web 2.0 egentlig står for og hvordan vi skal bruke det, men nå begynner det allerede å dukke opp referanser til Web 3.0...

For eksempel i denne artikkelen: Entrepreneurs See a Web Guided by Common Sense.

Det finnes ingen Web 3.0 side på WikiPedia - det er faktisk satt opp en sperre mot å opprette en slik side - men siden om Semantic Web kan være av interesse. Jeg skal også lese artikkelen Wikipedia 3.0: The End of Google? når jeg får litt tid.
